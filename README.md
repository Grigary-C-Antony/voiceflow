
# VoiceFlow 🎙️➡️📝

**AI-powered offline voice-to-text desktop app for Windows**

VoiceFlow is an open-source desktop application that converts speech to text **completely offline using AI** — no internet required.

Built as a weekend project to experiment with local AI, speech recognition, and desktop app development.

> ⚠️ Experimental Release  
> This project has only been tested on my personal Windows PC so far. It may contain bugs or compatibility issues. Feedback and contributions are welcome.

---

## ✨ Features

- 🎤 Voice to text transcription
- 📴 Fully offline (no internet required)
- 🤖 AI-powered local speech recognition
- 🔒 Privacy-friendly (audio stays on your device)
- ⚡ Fast transcription
- 🪟 Windows desktop application
- 🆓 Open source

## Installation (For Developers)

### Prerequisites

Install:

* Node.js
* Rust
* Tauri dependencies

### Clone Repository

```bash
git clone https://github.com/yourusername/voiceflow.git
cd voiceflow
```

### Install Dependencies

```bash
npm install
```

### Run in Development

```bash
npm run tauri dev
```

### Build Windows App

```bash
npm run tauri build
```

## Why VoiceFlow?

Most voice-to-text tools rely on cloud APIs.

VoiceFlow is built for:

✅ Offline usage
✅ Better privacy
✅ Open-source customization
✅ Local AI processing
